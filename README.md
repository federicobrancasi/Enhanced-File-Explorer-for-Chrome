# Enhanced File Explorer for Chrome

A beautiful, customizable file explorer that transforms Chrome's native `file://` directory listing into a modern, themed interface.

## Features

- **14 Built-in Themes** - Light, Dark, Ocean, Midnight, Forest, Sunset, Lavender, Rose, Sepia, Nord, High Contrast, and 3 dark variants
- **Custom Themes** - Create, name, and save unlimited custom color schemes
- **14 Font Options** - Sans-serif, monospace, and serif fonts from Google Fonts
- **Icon Styles** - Choose between SVG icons or emojis for files/folders
- **Quick Settings** - Floating toolbar for instant theme/font changes
- **Live Preview** - See changes in real-time in the settings page
- **Responsive Design** - Clean, modern interface that works on any screen

## Installation

### Chrome Web Store (Recommended)

1. Visit the [Chrome Web Store page](https://chromewebstore.google.com/detail/better-file-explorer-for/enoogpocakiocfjdghdahjnmpgejpghd)
2. Click "Add to Chrome"
3. **Important**: Enable "Allow access to file URLs" in extension settings

### Manual Installation (Development)

1. Clone this repository:
   ```bash
   git clone https://github.com/federicobrancasi/Enhanced-File-Explorer-for-Chrome.git
   ```
2. Open `chrome://extensions/` in Chrome
3. Enable "Developer mode" (top right)
4. Click "Load unpacked" and select the cloned folder
5. Enable "Allow access to file URLs" for the extension

## Usage

1. Navigate to any local directory using `file://` URL
2. The extension automatically enhances the page
3. Click the gear icon (top-right) for quick settings
4. Visit the extension options for full customization

## Browser Compatibility
